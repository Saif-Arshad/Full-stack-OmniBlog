## OmniBlog

OmniBlog is a platform that provides informative articles and blog posts about various aspects of life and has an admin dashboard where the admin can create, update, and delete blogs.
## Tech Stack Used 

Next JS  
JWT Authentication  
Uploadanything.com  
Tailwind Css   
Mongodb Atlas  
Restful API   
React Hot Toast
next-themes
